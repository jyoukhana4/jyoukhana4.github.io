Tetris on FPGA

(ECE 385) Digital Systems Laboratory Final Project

For my ECE 385 final project, I implemented a fully functional version of Tetris on an FPGA using the RealDigital Urbana board. This project showcased a hardware-software co-design approach, integrating a MicroBlaze softcore processor for USB keyboard input via SPI communication with a MAX3421E USB host controller. The game logic, including block spawning, movement, rotation, collision detection, and line-clearing, was implemented in SystemVerilog using a 10x22 grid structure. HDMI output was achieved through a VGA controller and a VGA-to-HDMI converter IP, with real-time display of game elements and on-screen ASCII-rendered text. A live score display and user interactions enhanced the gameplay, while timing and rendering were synchronized with video frame signals.
Beyond the core gameplay, I also incorporated an audio playback module that generated the classic Tetris theme using pulse-width modulation and a music ROM. This involved decoding encoded note and octave data to produce square wave audio signals compatible with the FPGAâ€™s AUX audio output. The overall system demonstrated modular design, combinational logic, SoC integration, and multimedia signal processing. This project highlighted my ability to design complex digital systems with real-time input/output handling, embedded audio, and efficient hardware resource management.
