Projects from coursework and personal

Reaction Wheel Pendulum

(ECE 486) Control Systems Project

Designed a real-time control system for a reaction wheel pendulum to achieve swing-up stabilization of a pendulum arm using the reaction torque generated by a DC motor driven flywheel. Designed and tested an observer-based controller to estimate unmeasured velocity states and improve real-time stabilization performance. Achieved swing-up, inverted stabilization, and switching control between downward and inverted equilibria on embedded hardware (TI-C2000 + Simulink) using friction compensation, state feedback, and observer-based state estimation


Low Dropout Regulator

(ECE 483) Analog IC Design Project

Designed a CMOS LDO designed in cadence 180nm technology, Delivering ~1.2V output voltage for a range of 0.1-10mA of load current. Implemented Folded-Cascode amplifier architecture with reference voltage and feedback as error amplifier. Achieved DC load regulation of 14uV/mA and DC line regulation of 60uV/V. minimal output noise, and loop-gain phase margin from 66.2 to 87 from a range of 0.1mA-10mA.


Tetris on FPGA

(ECE 385) Digital Systems Laboratory Final Project

Designed with system-on-chip MicroBlaze softcore processor, used to handle USB input. Capable of HDMI display, the game is controlled using a standard USB keyboard, as well as having AUDIO output through AUDIO OUT of FPGA board.


AC-DC Power supply w/ Amplification

(ECE 343) Electronic Circuits Laboratory

Designed and tested a regulated AC–DC power supply using a full-bridge rectifier, capacitor filter, Zener reference, op-amp voltage gain stage, and BJT current boost stage. The project focused on balancing cost, ripple performance, voltage regulation, load current capability, and efficiency.
